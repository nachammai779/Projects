## Overview

This folder contains assignments and exercises from an Information Systems course. The work is divided into two tracks: **Python** (Jupyter notebooks covering core programming concepts) and **R** (data analysis and statistical computing).

---

## Folder Structure

```
Information Systems/
├── Python/
│   ├── Built-in Data Types.ipynb
│   ├── Conditional and Loop.ipynb
│   ├── Dictionaries.ipynb
│   ├── Loops and Comprehensions.ipynb
│   ├── Strings and String Operations.ipynb
│   └── Assignment_4.ipynb
└── R/
```

---

## Python Notebooks

| Notebook | Topics Covered |
|----------|---------------|
| Built-in Data Types.ipynb | int, float, str, list, tuple, dict, set |
| Conditional and Loop.ipynb | if/elif/else, for loops, while loops |
| Dictionaries.ipynb | Creating, accessing, updating, iterating dictionaries |
| Loops and Comprehensions.ipynb | Nested loops, list/dict comprehensions |
| Strings and String Operations.ipynb | String methods, slicing, formatting |
| Assignment_4.ipynb | Comprehensive practical assignment covering all Python fundamentals |

---

## R Track

The R subfolder contains data analysis work using R, covering data frames, statistical summaries, and data visualization.

---

## Requirements

- Python 3.6+
- Jupyter Notebook or JupyterLab
- R 4.0+ (for R track)
