> Implementation of process scheduling algorithms in C for an Operating Systems course assignment.

---

## Overview

This project focuses on **process scheduling** concepts in operating systems. Scheduling algorithms determine the order in which processes receive CPU time, balancing throughput, response time, and fairness.

---

## Key Concepts

- CPU scheduling algorithms (FCFS, Round Robin, Priority)
- Process states: new, ready, running, waiting, terminated
- Context switching overhead
- Scheduling metrics: turnaround time, waiting time, response time
- Preemptive vs non-preemptive scheduling
